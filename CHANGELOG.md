# Changelog

## 0.1.0

### Phase 1.0
- Baseline FastAPI PDF RAG backend with upload, ingestion, retrieval, and grounded responses.
- ChromaDB vector persistence and sentence-transformer embeddings.

### Phase 1.1
- Section-aware chunking and parser text normalization improvements.
- Retrieval threshold support and stronger ingestion reliability.

### Phase 1.2A
- Token-aware context packing with tokenizer abstraction.
- Feature-flagged reranker contract and latency guardrail.
- Offline retrieval evaluator and improved API docs/examples.

### Phase 1.2B
- Deduplication and diversity-aware retrieval flow.
- Query-aware citation evidence snippets.
- Expanded retrieval analytics and report tooling.
